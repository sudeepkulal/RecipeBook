Here’s a **step-by-step copy-paste guide** to set up and run your Django project on a new laptop:

---

## 🚀 Setup Django Project on New Laptop

### 1. Go to your project folder

```bash
cd path/to/project
```

*(this is the folder where `manage.py` is located)*

---

### 2. Create Virtual Environment

```bash
python -m venv venv
```

---

### 3. Activate Virtual Environment

* **Windows (Command Prompt):**

  ```bash
  venv\Scripts\activate
  ```
* **Linux/Mac:**

  ```bash
  source venv/bin/activate
  ```

---

### 4. Upgrade pip (recommended)

```bash
pip install --upgrade pip
```

---

### 5. Install Django

```bash
pip install django
```

---

### 6. Run Migrations

```bash
python manage.py makemigrations
python manage.py migrate
```

---

### 7. (Optional) Create Superuser for Admin Panel

```bash
python manage.py createsuperuser
```

---

### 8. Run Development Server

```bash
python manage.py runserver
```

Now open 👉 **[http://127.0.0.1:8000/](http://127.0.0.1:8000/)** in your browser 🎉
